# v0.55 / Mission Run Live Gateway Health Strip 连接健康条

提交信息：`v0.55: Mission Run Live Gateway Health Strip 连接健康条`

## 版本分配依据

v0.11/v0.12 已有 Live Gateway 连接健康摘要，但只挂在 Gateway 会话区。Mission Run 首屏仍缺少一眼可见的 live/preflight/fallback/ping 状态。本轮只做 presentation-layer 健康条，复用既有 `ClawGatewayLiveHealthSummary`，不改 transport 协议。

## 目标

- 新增 `ClawMissionRunLiveGatewayHealthStrip`，从 `ClawGatewayLiveHealthSummary` 派生 title/status/guidance/chips。
- `ClawStore.missionRunLiveGatewayHealthStrip` 从现有 `gatewayLiveHealthSummary` 派生。
- compact Mission Run 与 iPad Dock 顶部展示；不执行连接动作，不写入 envelope。
- XCTest / LogicSmoke 覆盖 idle、fallback、streaming/ping、脱敏。
- 同步 README、flow、test、update_log。

## 非目标

- 不新增 schema/event/action/artifact kind。
- 不做真实心跳协议、后台保活、配对或自动重连策略变更。
- 不展示 raw endpoint query、Authorization header、token 原文。

## 验收标准

- 健康条只读既有 summary 字段。
- 敏感字符串不外显。
- 云端 CI artifact 验收通过。
